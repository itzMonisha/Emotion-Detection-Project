# Emotion Detection using Deep Learning

## Overview
This project detects human emotions from facial images using a Convolutional Neural Network (CNN).

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- Jupyter Notebook

## Features
- Emotion classification
- Image preprocessing
- CNN-based prediction

## Dataset
## Dataset

This project uses the FER2013 dataset.
